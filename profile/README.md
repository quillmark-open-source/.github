# Quillmark Open Source

Open-source tools for grounded AI workflows.

Quillmark Open Source is an initiative from Quillmark LLC. We publish software that helps teams turn sources, evidence, review criteria, and follow-up steps into workflows they can repeat and inspect.

The work is meant for places where context matters: regulated processes, document-heavy review, operational handoffs, internal tools, and agent workflows that need clear evidence trails.

## What We Build

- Codex plugins and skills
- source-backed workflow packages
- small utilities for capture, evidence preservation, and review support
- examples and recipes for putting tools together in practical workflows
- documentation for teams that want to inspect, adapt, and run the work themselves

## Projects

### Compliance Flag

[Compliance Flag](https://github.com/quillmark-open-source/compliance-flag) is the first Quillmark Open Source project. It is an open-source Codex plugin and workflow package for evidence-backed SEC Marketing Rule website review.

It helps teams collect public RIA website pages, preserve page evidence, and use Codex to draft findings for qualified human review.

Compliance Flag is a review-support tool. It does not provide legal advice, compliance advice, regulatory approval, guaranteed compliance, or final review decisions.

## How To Participate

Quillmark LLC maintains these projects, and community participation is welcome.

Good ways to get involved:

- open issues with clear bug reports, questions, or workflow ideas
- improve documentation, examples, and recipes
- test projects in real development environments
- propose focused changes through pull requests
- share use cases that would make the public projects more useful

Early projects are maintainer-led. Maintainer roles and governance may expand as the contributor community grows.
